## Python

* Version : 3.10.9

## Install

### Windows

```sh
pip install pandas
pip install sklearn
pip install sentence_transformers
```

### Mac / Linux

```sh
pip3 install pandas
pip3 install sklearn
pip3 install sentence_transformers
```
